FAQs

Apa itu Corelyttics ?

Corelytics adalah layanan data analytics untuk UMKM yang membantu Anda memahami data penjualan, memprediksi permintaan, dan mengelompokkan pelanggan secara cerdas.

Apakah saya perlu paham data untuk menggunakan layanan ini ?

Tidak perlu. Layanan kami dirancang untuk mudah digunakan bahkan oleh pemilik bisnis tanpa latar belakang data.

Berapa biaya menggunakan Corelytics?

Kami menyediakan beberapa paket yang fleksibel sesuai kebutuhan UMKM Anda. Silakan cek bagian harga untuk detail lebih lanjut

Layanan apa saja yang ditawarkan Corelytics?

Corelytics menawarkan layanan seperti Analisis Data Penjualan, Visualisasi Interaktif, Prediksi Permintaan Produk, dan Segmentasi Pelanggan untuk membantu UMKM mengambil keputusan yang lebih cerdas dan tepat waktu.

Seberapa cepat saya bisa melihat hasil dari layanan Corelytics?

Anda dapat mulai melihat insight awal dalam waktu kurang dari 24 jam setelah data Anda diproses. Untuk analisis yang lebih mendalam seperti prediksi dan segmentasi, biasanya memerlukan waktu 2â€“3 hari kerja.

Apakah layanan Corelytics bisa disesuaikan dengan jenis usaha saya?

Ya, layanan kami dirancang fleksibel dan dapat disesuaikan dengan jenis dan skala usaha Anda. Kami akan menyesuaikan analisis berdasarkan data bisnis yang Anda miliki.

Data saya dalam format Excel, apakah bisa digunakan?

Tentu, Corelytics mendukung berbagai format data termasuk Excel (.xlsx), CSV, dan Google Sheets. Anda hanya perlu mengunggah file melalui dashboard atau menghubungkan akun Google Drive Anda.

Apakah saya perlu menginstal software khusus untuk menggunakan Corelytics?

Tidak perlu. Corelytics sepenuhnya berbasis web, jadi Anda cukup mengakses layanan melalui browser tanpa instalasi tambahan.

Bagaimana jika data saya mengalami perubahan? Haruskah saya mengunggah ulang?

Jika Anda menggunakan unggahan manual, ya, Anda perlu mengunggah ulang file terbaru. Namun jika Anda menggunakan integrasi otomatis (seperti Google Sheets atau API), data akan diperbarui secara otomatis sesuai jadwal sinkronisasi.

Apakah data saya aman di Corelytics?

Ya, keamanan data adalah prioritas kami. Corelytics menggunakan enkripsi standar industri (SSL/TLS) untuk melindungi data selama transmisi dan penyimpanan.

Siapa saja yang bisa mengakses data saya?

Hanya Anda dan anggota tim yang Anda beri akses melalui dashboard. Kami tidak akan membagikan atau melihat data Anda tanpa izin.

Apakah Corelytics mematuhi regulasi perlindungan data?

Ya, kami mematuhi standar perlindungan data seperti GDPR dan aturan lokal terkait privasi data pengguna di Indonesia.
